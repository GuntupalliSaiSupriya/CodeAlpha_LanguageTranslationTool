# CodeAlpha_LanguageTranslationTool

## Project Description
This is a Language Translation Tool developed as part of the CodeAlpha Artificial Intelligence Internship.

## Features
- Translate text between multiple languages
- User-friendly interface
- Supports English, Telugu, Hindi, Tamil, Kannada, Malayalam, French, German, and Spanish

## Technologies Used
- Python
- Streamlit
- Deep Translator

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py
